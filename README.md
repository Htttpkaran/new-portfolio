# Karan Kumar - Personal Portfolio

Welcome to the source code for my personal portfolio website!

## Overview
This is a modern, responsive portfolio built with React and Vite. It showcases my projects, skills, and experience as a developer.

## Technologies Used
- React
- Vite
- Custom CSS

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
